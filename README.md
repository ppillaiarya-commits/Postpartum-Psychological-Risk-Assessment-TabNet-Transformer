Postpartum Psychological Risk Assessment using Hybrid TabNet–Transformer Encoder
Overview

This repository contains the implementation of a Hybrid TabNet–Transformer Encoder framework for questionnaire-based postpartum psychological risk assessment using maternal psychological indicators.

Model Architecture

TabNet Encoder → Transformer Encoder → Classification Layer

Features
Attentive Feature Selection
Transformer-based Contextual Learning
Feature Importance Analysis
ROC Curve Evaluation
Accuracy and Loss Monitoring
Stratified Cross-Validation

Objective

The proposed framework aims to identify anxiety-associated postpartum psychological risk using questionnaire-based maternal psychological indicators through attentive feature learning and contextual representation modeling.

Results

The Hybrid TabNet–Transformer Encoder achieved strong classification performance with stable convergence behavior, robust generalization capability, and interpretable feature importance analysis.
